## Overview

The `AdjacencyMatrixGraph` class represents an unweighted graph using an adjacency matrix. It supports both directed and undirected graphs and stores edges as boolean values in a 2D matrix.

## Constructor

```python
AdjacencyMatrixGraph(directed=False, weighted=False, vertices=None)
```

<ParamField path="directed" type="bool" default="False">
  Whether the graph is directed
</ParamField>

<ParamField path="weighted" type="bool" default="False">
  Whether the graph is weighted (always False for this class)
</ParamField>

<ParamField path="vertices" type="list[str]" default="None">
  List of vertex labels to initialize the graph with
</ParamField>

**Raises:** `ValueError` if duplicate vertex labels are provided

```python
from dsa.graph import AdjacencyMatrixGraph

# Create empty graph
g = AdjacencyMatrixGraph()

# Create directed graph
g = AdjacencyMatrixGraph(directed=True)

# Create with initial vertices
g = AdjacencyMatrixGraph(vertices=['A', 'B', 'C', 'D'])
```

```python Graph Operations
from dsa.graph import AdjacencyMatrixGraph

# Create graph from dictionary
data = {
    'A': ['B', 'C'],
    'B': ['A', 'D'],
    'C': ['A', 'D'],
    'D': ['B', 'C']
}
g = AdjacencyMatrixGraph.from_dict(data)

# Perform operations
print(f"Initial vertices: {g.vertices()}")
print(f"Initial edges: {g.size()}")

# Add new vertex and edges
g.add_vertex('E')
g.add_edge('D', 'E')
g.add_edge('E', 'A')

print(f"After addition: {g.vertices()}")
print(f"After addition: {g.size()}")

# Remove edge
g.delete_edge('A', 'B')
print(f"After removing A-B: {g.size()}")

# Remove vertex (removes all connected edges)
g.delete_vertex('E')
print(f"After removing E: {g.vertices()}")
print(f"After removing E: {g.size()}")

# Convert back to dictionary
print(f"Final graph: {g.to_dict()}")
```

```python Graph Traversal
from dsa.graph import AdjacencyMatrixGraph
from collections import deque

# Create graph
g = AdjacencyMatrixGraph(vertices=['A', 'B', 'C', 'D', 'E'])
g.add_edge('A', 'B')
g.add_edge('A', 'C')
g.add_edge('B', 'D')
g.add_edge('C', 'D')
g.add_edge('D', 'E')

# BFS traversal
def bfs(graph, start):
    visited = set()
    queue = deque([start])
    result = []
    
    while queue:
        vertex = queue.popleft()
        if vertex not in visited:
            visited.add(vertex)
            result.append(vertex)
            for neighbor in graph.adjacents(vertex):
                if neighbor not in visited:
                    queue.append(neighbor)
    
    return result

print(f"BFS from A: {bfs(g, 'A')}")

# DFS traversal
def dfs(graph, start, visited=None):
    if visited is None:
        visited = set()
    
    visited.add(start)
    result = [start]
    
    for neighbor in graph.adjacents(start):
        if neighbor not in visited:
            result.extend(dfs(graph, neighbor, visited))
    
    return result

print(f"DFS from A: {dfs(g, 'A')}")
```

## Performance

- **Space Complexity:** O(V²) where V is the number of vertices
- **Add Vertex:** O(V²) - requires resizing matrix
- **Delete Vertex:** O(V²) - requires resizing matrix
- **Add Edge:** O(1)
- **Delete Edge:** O(1)
- **Has Edge:** O(1)
- **Get Adjacents:** O(V)
- **Get All Edges:** O(V²)

## Notes

- Best for dense graphs where most vertices are connected
- Provides O(1) edge lookup time
- Uses more space than adjacency list for sparse graphs
- Vertex labels must be strings
- Self-loops are not supported
- For undirected graphs, edges are stored symmetrically in the matrix
