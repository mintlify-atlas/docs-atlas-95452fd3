## Overview

The `AdjacencyMatrixWeightedGraph` class represents a weighted graph using an adjacency matrix. It extends `AdjacencyMatrixGraph` and stores edge weights as numeric values in a 2D matrix. Supports both directed and undirected graphs.

## Constructor

```python
AdjacencyMatrixWeightedGraph(directed=False, vertices=None)
```

<ParamField path="directed" type="bool" default="False">
  Whether the graph is directed
</ParamField>

<ParamField path="vertices" type="list[str]" default="None">
  List of vertex labels to initialize the graph with
</ParamField>

**Raises:** `ValueError` if duplicate vertex labels are provided

```python
from dsa.graph import AdjacencyMatrixWeightedGraph

# Create empty weighted graph
g = AdjacencyMatrixWeightedGraph()

# Create directed weighted graph
g = AdjacencyMatrixWeightedGraph(directed=True)

# Create with initial vertices
g = AdjacencyMatrixWeightedGraph(vertices=['A', 'B', 'C'])
```

```python Network Latency
from dsa.graph import AdjacencyMatrixWeightedGraph
import sys

# Create network with latency in ms
g = AdjacencyMatrixWeightedGraph(directed=True)

# Add connections with latencies
g.add_edge('Server1', 'Server2', 10)
g.add_edge('Server1', 'Server3', 25)
g.add_edge('Server2', 'Server4', 15)
g.add_edge('Server3', 'Server4', 5)
g.add_edge('Server4', 'Server5', 20)

# Dijkstra's shortest path algorithm
def dijkstra(graph, start, end):
    vertices = graph.vertices()
    distances = {v: sys.maxsize for v in vertices}
    distances[start] = 0
    visited = set()
    previous = {v: None for v in vertices}
    
    while len(visited) < len(vertices):
        # Find unvisited vertex with minimum distance
        current = None
        min_dist = sys.maxsize
        for v in vertices:
            if v not in visited and distances[v] < min_dist:
                min_dist = distances[v]
                current = v
        
        if current is None or current == end:
            break
            
        visited.add(current)
        
        # Update distances to neighbors
        for neighbor, weight in graph.adjacent_items(current):
            if neighbor not in visited:
                new_dist = distances[current] + weight
                if new_dist < distances[neighbor]:
                    distances[neighbor] = new_dist
                    previous[neighbor] = current
    
    # Reconstruct path
    path = []
    current = end
    while current is not None:
        path.insert(0, current)
        current = previous[current]
    
    return path, distances[end]

path, latency = dijkstra(g, 'Server1', 'Server5')
print(f"Shortest path: {' -> '.join(path)}")
print(f"Total latency: {latency} ms")
```

```python Minimum Spanning Tree
from dsa.graph import AdjacencyMatrixWeightedGraph

# Create graph for MST
g = AdjacencyMatrixWeightedGraph()

# Add edges with costs
g.add_edge('A', 'B', 4)
g.add_edge('A', 'C', 2)
g.add_edge('B', 'C', 1)
g.add_edge('B', 'D', 5)
g.add_edge('C', 'D', 8)
g.add_edge('C', 'E', 10)
g.add_edge('D', 'E', 2)
g.add_edge('D', 'F', 6)
g.add_edge('E', 'F', 3)

# Kruskal's MST algorithm (simplified)
def find_mst(graph):
    edges = sorted(graph.undirected_edges(), key=lambda x: x[2])
    parent = {v: v for v in graph.vertices()}
    
    def find(v):
        if parent[v] != v:
            parent[v] = find(parent[v])
        return parent[v]
    
    def union(v1, v2):
        root1 = find(v1)
        root2 = find(v2)
        parent[root1] = root2
    
    mst = []
    total_cost = 0
    
    for start, end, weight in edges:
        if find(start) != find(end):
            union(start, end)
            mst.append((start, end, weight))
            total_cost += weight
    
    return mst, total_cost

mst, cost = find_mst(g)
print("Minimum Spanning Tree:")
for start, end, weight in mst:
    print(f"  {start} - {end}: {weight}")
print(f"Total cost: {cost}")
```

## Performance

- **Space Complexity:** O(V²) where V is the number of vertices
- **Add Vertex:** O(V²) - requires resizing matrix
- **Delete Vertex:** O(V²) - requires resizing matrix
- **Add Edge:** O(1)
- **Delete Edge:** O(1)
- **Has Edge:** O(1)
- **Get Weight:** O(1)
- **Get Adjacents:** O(V)
- **Get All Edges:** O(V²)

## Notes

- Best for dense graphs where most vertices are connected
- Provides O(1) edge weight lookup
- Weights can be any numeric type (int, float, etc.)
- Self-loops are prevented (diagonal is set to None)
- For undirected graphs, edge weights are stored symmetrically
- Vertex labels must be strings
- Useful for shortest path algorithms (Dijkstra, Floyd-Warshall)
- Ideal for minimum spanning tree algorithms (Prim, Kruskal)
