## Overview

The `AdjacencyListGraph` class represents an unweighted graph using an adjacency list. It stores each vertex and its list of adjacent vertices in a dictionary, making it space-efficient for sparse graphs.

## Constructor

```python
AdjacencyListGraph(directed=False, vertices=None)
```

<ParamField path="directed" type="bool" default="False">
  Whether the graph is directed
</ParamField>

<ParamField path="vertices" type="list[str]" default="None">
  List of vertex labels to initialize the graph with
</ParamField>

```python
from dsa.graph import AdjacencyListGraph

# Create empty graph
g = AdjacencyListGraph()

# Create directed graph
g = AdjacencyListGraph(directed=True)

# Create with initial vertices
g = AdjacencyListGraph(vertices=['A', 'B', 'C', 'D'])
```

# Topological sort (simplified)
def topological_sort(graph):
    in_degree = {v: 0 for v in graph.vertices()}
    for v in graph.vertices():
        for neighbor in graph.adjacents(v):
            in_degree[neighbor] += 1
    
    queue = [v for v in graph.vertices() if in_degree[v] == 0]
    result = []
    
    while queue:
        vertex = queue.pop(0)
        result.append(vertex)
        
        for neighbor in graph.adjacents(vertex):
            in_degree[neighbor] -= 1
            if in_degree[neighbor] == 0:
                queue.append(neighbor)
    
    return result

order = topological_sort(g)
print(f"Task execution order: {order}")

```python Graph Algorithms
from dsa.graph import AdjacencyListGraph
from collections import deque

# Create graph
g = AdjacencyListGraph()
g.add_edge('A', 'B')
g.add_edge('A', 'C')
g.add_edge('B', 'D')
g.add_edge('C', 'D')
g.add_edge('D', 'E')
g.add_edge('C', 'F')

# BFS - shortest path in unweighted graph
def bfs_shortest_path(graph, start, end):
    if start == end:
        return [start]
    
    visited = {start}
    queue = deque([(start, [start])])
    
    while queue:
        vertex, path = queue.popleft()
        
        for neighbor in graph.adjacents(vertex):
            if neighbor not in visited:
                new_path = path + [neighbor]
                if neighbor == end:
                    return new_path
                visited.add(neighbor)
                queue.append((neighbor, new_path))
    
    return None

path = bfs_shortest_path(g, 'A', 'E')
print(f"Shortest path A to E: {' -> '.join(path)}")

# Find all paths
def find_all_paths(graph, start, end, path=[]):
    path = path + [start]
    if start == end:
        return [path]
    
    paths = []
    for neighbor in graph.adjacents(start):
        if neighbor not in path:
            new_paths = find_all_paths(graph, neighbor, end, path)
            paths.extend(new_paths)
    
    return paths

all_paths = find_all_paths(g, 'A', 'E')
print(f"All paths A to E:")
for p in all_paths:
    print(f"  {' -> '.join(p)}")
```

```python Connected Components
from dsa.graph import AdjacencyListGraph

# Create disconnected graph
g = AdjacencyListGraph()

# Component 1
g.add_edge('A', 'B')
g.add_edge('B', 'C')

# Component 2
g.add_edge('D', 'E')
g.add_edge('E', 'F')

# Component 3 (isolated)
g.add_vertex('G')

# Find connected components
def find_components(graph):
    visited = set()
    components = []
    
    def dfs(vertex, component):
        visited.add(vertex)
        component.append(vertex)
        for neighbor in graph.adjacents(vertex):
            if neighbor not in visited:
                dfs(neighbor, component)
    
    for vertex in graph.vertices():
        if vertex not in visited:
            component = []
            dfs(vertex, component)
            components.append(component)
    
    return components

components = find_components(g)
print(f"Number of components: {len(components)}")
for i, comp in enumerate(components, 1):
    print(f"Component {i}: {comp}")
```

## Performance

- **Space Complexity:** O(V + E) where V = vertices, E = edges
- **Add Vertex:** O(1)
- **Delete Vertex:** O(V + E) - must remove vertex from all adjacency lists
- **Add Edge:** O(1) average case
- **Delete Edge:** O(degree(V)) - must search adjacency list
- **Has Edge:** O(degree(V))
- **Get Adjacents:** O(1)
- **Get All Edges:** O(V + E)

## Notes

- Best for sparse graphs where E is much less than V²
- More space-efficient than adjacency matrix for sparse graphs
- Faster vertex addition than adjacency matrix
- Slower edge lookup than adjacency matrix
- Vertex labels must be strings
- Adjacency lists are implemented using Python dictionaries and lists
- For undirected graphs, edges are stored in both directions
